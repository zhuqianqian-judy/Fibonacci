# Fibonacci
============

实现Fibonacci数列：要求（以java为例）如下： <br>
函数名Fibonacci.of()                      <br>
of是一个静态方法，入参是int,返回值是int     <br>
参数从1开始，各参考值如下：                 <br>
    Fibonacci.of(1) == 1                  <br>
    Fibonacci.of(2) == 1                  <br>
    Fibonacci.of(3) == 2                  <br>
    Fibonacci.of(4) == 3                  <br>
    Fibonacci.of(5) == 5                  <br>
    ...                                   <br>
在main()函数用循环打印1-200的Fibonacci数列
